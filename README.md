# Timer for Life – Privacy Policy (Vercel)

This folder is a simple static website containing the Privacy Policy for the **Timer for Life** mobile app.

## Files

- `index.html`: privacy policy page
- `styles.css`: page styling
- `vercel.json`: Vercel static deploy configuration

## Deploy to Vercel (no build step)

### Option A: Import a Git repository

1. Push this folder to GitHub (recommended as its own repo).
2. In Vercel, click **New Project** → **Import** your repo.
3. Framework preset: **Other**
4. Build Command: **None**
5. Output Directory: **.**
6. Deploy

### Option B: Drag & Drop

1. Go to Vercel → **Add New** → **Project**
2. Drag and drop this folder contents (`index.html`, `styles.css`, `vercel.json`)
3. Deploy
